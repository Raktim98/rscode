An interactive and user-friendly dictionary web application built using HTML, CSS, and JavaScript.
The project allows users to search for word meanings, pronunciations, examples, synonyms, and more using a dictionary API.

🚀 Features
🔍 Search meanings of words instantly
🎤 Voice search support using microphone input
🌙 Dark mode support
🔊 Audio pronunciation of words
📖 Word definitions with examples
🧠 Synonyms and antonyms
📱 Responsive design for mobile and desktop
⚡ Fast and simple user interface
🛠️ Technologies Used
HTML5
CSS3
JavaScript (ES6)
Dictionary API
